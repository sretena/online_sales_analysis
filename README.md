# Online Sales Analysis

## Opis projekta

Ovaj projekat predstavlja jednostavan sistem za upravljanje proizvodima i korpom kupca, razvijen u Python-u. Za verzionisanje projekta korišćen je Git i GitHub.

## Klase

### Product
- čuva naziv proizvoda
- cenu proizvoda
- količinu proizvoda
- prikazuje informacije o proizvodu
- ažurira količinu proizvoda

### ProductManager
- dodaje proizvode
- uklanja proizvode
- prikazuje sve proizvode
- računa ukupnu vrednost inventara

### Cart
- dodaje proizvode u korpu
- prikazuje sadržaj korpe
- računa ukupnu vrednost korpe

## Funkcionalnosti

- Dodavanje proizvoda
- Uklanjanje proizvoda
- Prikaz svih proizvoda
- Izračunavanje ukupne vrednosti inventara
- Upravljanje korpom kupca
- Izračunavanje ukupne vrednosti korpe

## Git funkcionalnosti

- Kreiranje i kloniranje repozitorijuma
- Rad sa granama (branches)
- Commit izmene
- Spajanje grana (merge)
- Korišćenje `.gitignore`
- Slanje izmena na GitHub (push)
